规则形式如下：


<target> : <prerequisites>

[tab] <commands>

目标：前置条件

目标是必须的，不可省略

前置条件， 命令都是可选的。但两者必须至少存在一个。

每条规则明确两件事情：构建目标的前置条件是什么，以及如何构建。


目标 target

目标通常是文件名，指明Make命令所构建的对象。比如上文的a.txt

目标可以是一个文件名，也可以是多个文件名，之间用空格分割。

除了文件名， 目标还可以是某个操作的名字， 伪目标（phony target）

```
clean: 
    rm *.o
```


前置条件：prerequisites

前置条件通常是一组文件名，之间用空格分割。

它指定了“目标”是否需要重建的判断标准
