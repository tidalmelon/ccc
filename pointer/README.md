1. 指针的类型：把声明语句的指针名去掉。
```
int*ptr : 指针的类型是。int*
char*ptr; : 指针的类型是 char*
int**ptr; : 指针的类型是 int**
int(*ptr)[3]; : 指针的类型是 int(*)[3]
int*(*ptr)[4]; : 指针的类型是 int*(*)[4]
```
